First:

npm install
npm install tensorflow
npm install bootstrap
npm install face-api
npm install seedrandom
npm install tslib

To run this project you will need to run it on a live server, via Atom or Visual Studio.
After doing the above:

Navigate to Project/views/index.html to run for live server.

The folder called js contains all relevant javascript, including the main script called index.js

The folder called models contains all faceapi models for drawing dimensions on the user's face.

The folder called project video contains a short demonstration of the project.

The folder called prototypes contains testing of separate systems of the project.

The folder called research contains images related the project from which I drew influence.

The folder called views contains the relevant html and css files for styling.

A webcam is required for this application to run at it's full potential.
